This command line application was created to provide a simple method of tracking the progress of each player in a game of \char`\"{}\+Catan\char`\"{} by Klaus Teuber.

Currently, the base game and \char`\"{}\+Catan\+: 5-\/6 Player Extension\char`\"{} are supported.\hypertarget{index_autotoc_md2}{}\doxysection{Benefits}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Quickly retrieve everyone\textquotesingle{}s victory points
\item Visualize distribution and test normality of dice rolls
\item Remind yourself of the last roll if you forget
\item Save everyone\textquotesingle{}s modus operandi at the end of the game
